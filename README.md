# nodejs-test
To test nodejs with mongodb

Environment variables for connecting to mongodb used in this app:

MONGODB_USERNAME 
MONGODB_PASSWORD
MONGODB_HOST
MONGODB_PORTMONGODB_DBNAME

Starting the app:

$ MONGODB_USERNAME=user1 MONGODB_PASSWORD=123 ENV_VAR=VALUE...  node app.js
